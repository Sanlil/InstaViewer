About

Usage

InstaViewer is a web application that allows you to look at and search for instagram pictures in your browser without any installation or login needed. You can both search by hashtags and usernames and then filter your result by additional words. There is 20 pictures loaded in one search and you can fetch more by scrolling to the end of the page.

In the top navigation bar you can find a switch between grid mode, with thumbnail pictures, and list mode with full sized pictures. There is also a button for updating the search. In grid mode you also have the possibility to click on a picture to see it in full size with all related information.

Competitors

Of course there is many similar pages out there. Because Instagram is a mobile first application, the web app has not all the functionality that you would like and a lot of people have therefore made their own versions. Two examples are gramfeed.com and pinsta.me that both have the normal Instagram functionality but in a web application. Other more extreme examples are ink361.com and iconosquare.com that has additional functionality as for example organizing photos and get statistics.

Nevertheless, I believe that the advantage of InstaViewer is its simplicity and for example that you don't need an Instagram account to access the page. Therefore, If I expand this application, I will continue in the same way. One possible extension would for example be to implement functionality to search for locations. Maybe even include a map were you can pinpoint a location and get pictures taken in that area.

The Development

InstaViewer is a single page app based on the Instagram API and developed in AngularJS and the JavaScript library JQuery. The application was made as a project in the course DV1483 JavaScript, jQuery and AJAX with HTML5 and PHP at Blekinge Institute of Technology.

The source code is available on GitHub and the course page on student.bth.se/~salp10/js/.
